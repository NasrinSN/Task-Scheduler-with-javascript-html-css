# Task Scheduler

## Overview

The Task Scheduler project demonstrates skills in HTML, CSS, and JavaScript. It is a web-based application that allows users to manage and schedule tasks efficiently. This project showcases the use of front-end technologies to create an interactive and user-friendly task management tool.

## Features

- Task Management: Add, edit, and delete tasks.
- Task Scheduling: Schedule tasks for specific times and dates.
- User Interface: Intuitive and responsive design using HTML and CSS.
- Interactivity: Dynamic task handling and scheduling using JavaScript.

## Project Structure

The project folder contains the following files and directories:

- `index.html`: The main HTML file for the application.
- `README.md`: This file with information about the project.
- `assets/`: Contains images and other static assets used in the project.
- `scripts/`: Contains JavaScript files that handle the functionality of the task scheduler.
- `styles/`: Contains CSS files for styling the application.

## Installation

To view and interact with the Task Scheduler application, follow these steps:

1. Clone the Repository:
   ```bash
   git clone https://github.com/NasrinSN/task-scheduler.git
   ```
2. Navigate to the Project Directory:
   ```bash
   cd task-scheduler
   ```
3. Open the `index.html` File:
   Open `index.html` in your web browser to view the application.

## Usage

- Add a Task: Enter task details in the provided input fields and click the "Add Task" button.
- Edit a Task: Click on a task to edit its details.
- Delete a Task: Click the "Delete" button next to a task to remove it from the list.

## Technologies Used

- HTML: Markup language for structuring the web pages.
- CSS: Styling language for designing the user interface.
- JavaScript: Programming language for adding interactivity and functionality.
